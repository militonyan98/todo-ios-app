# TODO APPLICATION - iOS SWIFT UI


## Main functionality

* Add Tasks
* Delete Tasks
* Edit Tasks
* View Tasks (sort by date, name, priority)

### *Each task has the following properties*

* Name
* Description
* Priority
* Date


## Technologies Used

* SwiftUI
* Core Data
* MVVM
