

# ![App Icon](https://github.com/militonyan98/todo-ios-app/blob/main/icon.jpeg?raw=true) TODO APPLICATION - iOS SWIFT UI


## Main functionality

* Add Tasks
* Delete Tasks
* Edit Tasks
* View Tasks (sort by date, name, priority)

### *Each task has the following properties*

* Name
* Description
* Priority
* Date


## Technologies Used

* SwiftUI
* Core Data
* MVVM
